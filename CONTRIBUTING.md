# Contributing

このテンプレートは複数リポジトリへ展開される前提のため、更新方針を明示します。

## 更新方針（テンプレート → 各リポジトリ）
1. テンプレート本体の変更は、必ずこのリポジトリに反映する。
2. 各リポジトリへの反映は、以下のいずれかで行う。
   - 差分適用（推奨）: テンプレートの変更を手動で反映
   - 再生成: テンプレートを再適用して差分を取り込み
3. 反映時は、`docs/template-checklist.md` に従って確認する。

## 変更レポート
要件/設計の見直しが発生した場合は `docs/change-report-template.md` を使用し、
`docs/reports/` に保存する。
