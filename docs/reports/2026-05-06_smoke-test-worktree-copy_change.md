# 変更レポート（要件/設計見直し）

## 概要
- 発生日: 2026-05-06
- 対象機能/モジュール: `scripts/smoke-test-generated.sh`
- 影響範囲: 生成後プロジェクト相当のローカルスモークテスト
- 関係者: AI作業者

## 1. 何が想定外で起きたのか
- スモークテストが `git archive HEAD` を使っていたため、ローカルで修正中の追跡済みファイルが検証対象に含まれなかった。

## 2. 想定外が起きた原因
- 未追跡ファイルを混入させない再現性を優先し、コミット済み状態だけを展開する実装にしていた。

## 3. 対応方法
- `git ls-files` と `tar` を使い、Git管理下の現在の作業ツリーを一時ディレクトリへコピーする方式に変更した。
- 未追跡ファイルはコピー対象に含めず、追跡済みの未コミット変更は検証対象に含める。

## 4. 要件定義や設計で修正した部分
- README に、ローカル実行時のスモークテストは追跡済み作業ツリーを検証することを明記した。
